## Code for HL

```py
number = int(input("Input a number: "))
factor = 2
print(1)
while factor <= number//2:
    if number%factor==0:
        print(factor)
    factor+=1

num = int(number)
digit_1 = int(num%10)
num_2 = int(num/10)
digit_2 = int(num_2%10)

if num != digit_1 + digit_2:
    print(f"False")
else:
    print(f"True")

```
